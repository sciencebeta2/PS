# 1001 : [기초-출력] 출력하기01(설명)

C/C++언어에서 가장 기본적인 명령이 출력문이다.
<br />
`printf()`를 이용해 다음 단어를 출력하시오.

Hello

참고
<br />
C/C++ IDE 프로그램을 설치한 후,
<br />
아래와 같은 소스 코드를 작성하고 "컴파일-실행"을 수행시키면,
<br />
지정한 "문장"이 출력된다.

```
#include <stdio.h>

int main()
{
  printf("문장");
  return 0;
}
```
