# [1014 : [기초-입출력] 문자 2개 입력받아 순서 바꿔 출력하기(설명)](https://codeup.kr/problem.php?id=1014)

2개의 문자(ASCII CODE)를 입력받아서 순서를 바꿔 출력해보자.

참고
```C
char x, y;
scanf("%c %c", &x, &y);
printf("%c %c", y, x); //출력되는 순서를 작성
```
와 같은 방법으로 해결할 수 있다.

--------------------------------------------------------------
- ### 입력/출력
  - #### 입력

    2개의 문자가 공백으로 구분되어 입력된다.

  - #### 출력

    두 문자의 순서를 바꿔 출력한다.

- ### 입력/출력 예시
  - #### 입력
    ```
    A b
    ```
  - #### 출력
    ````
    b A
    ````
